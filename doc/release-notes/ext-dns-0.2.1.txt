Advanced Software Production Line is proud to announce the first
public release of ext-Dns toolkit.

ext-Dns is a ready to use toolkit that will allow you building DNS
servers that performs additional operations when a request is
received.

This release in short
~~~~~~~~~~~~~~~~~~~~~
  
  First public production ready stable release. This first release
  includes a base core library with all DNS functions and a ready to
  use caching-forward DNS server that delegates policy resolution to a
  child resolver that can be written in any language.
